eslint

注：这里不讲eslint的知识，仅仅讲使用

一、安装(以下是全局的，同时安装到项目中--save-dev)

  1. npm install eslint -g
  2. npm install eslint-plugin-import -g
  3. npm install eslint-plugin-react -g
  4. npm install babel-eslint -g
